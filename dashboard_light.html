⚙️ Technical Overview

Application Purpose:
This dashboard provides real-time atmospheric telemetry for cities across Portugal. By hovering over any city card, you can view precise geographical coordinates alongside the current maximum and minimum temperatures at that exact moment.

Architectural Stack:
Built natively in C++20 on Linux, the backend operates on a strict multi-threaded architecture. It utilizes libcurl for non-blocking network I/O and the Crow framework for high-concurrency REST API delivery.

Thread Safety & Lifecycle:
The background telemetry ingest is completely decoupled from the API serving layer via std::atomic signals and a locked std::mutex cache. This ensures zero-latency reads for incoming HTTP requests and proper resource cleanup during graceful shutdowns. The entire microservice is containerized via Docker for lightweight, environment-agnostic deployment.
